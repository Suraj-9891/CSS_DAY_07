These Shadow is Box-shadow?

CSS Shadow
Box-Shadow Adds a shadow to the box model of an element
Syntax: box-shadow: none| [inset] || [length] || [length] || [length] || [color]
Spread radius: Expands or contracts the shadow.
Color: Sets the shadow color.
1. Horizontal Offset: Moves the shadow horizontally.
2. Vertical Offset: Moves the shadow vertically.
3. Blur Radius: Blurs the shadow.
4. Spread Radius: Expands or contracts the shadow.
5. Color: Sets the shadow color.
