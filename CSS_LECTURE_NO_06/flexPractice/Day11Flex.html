What is Flexbox:-Flexbox, short for Flexible Box, is a CSS layout model that enables you to create flexible and dynamic layouts
1.Flex Container: The container element that holds the flex items.
2.Flex Items: The child elements of the flex container.
3.Main Axis: The primary axis of the flex container (horizontal or vertical).
4.Cross Axis: The secondary axis of the flex container (perpendicular to the main axis).
5.Flex Direction: The direction of the flex items along the main axis.
